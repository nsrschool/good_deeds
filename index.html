📖 คู่มือการใช้งานระบบบันทึกความดี

  1. ไปที่ ⚙️ ตั้งค่าระบบ → ข้อมูลสถานศึกษา กรอกชื่อโรงเรียน ปีการศึกษา ผู้อำนวยการ และอัปโหลดโลโก้/ลายเซ็น แล้วกดบันทึก
  2. ไปที่ ⚙️ ตั้งค่าระบบ → รายชื่อครู เพิ่มรายชื่อครูผู้บันทึกทุกคน (หรือกดปุ่ม ➕ ที่หน้าบันทึกความดีก็ได้)
  3. ไปที่ ⚙️ ตั้งค่าระบบ → จัดการรายชื่อนักเรียน ดาวน์โหลดไฟล์ฟอร์มตัวอย่าง กรอกรายชื่อ แล้วนำเข้าไฟล์ (.xlsx/.csv) หรือเพิ่มรายชื่อทีละคนก็ได้
  4. ไปที่ 📝 บันทึกความดี เลือกชั้น เช่น ป.1/1 → เลือกนักเรียน (แสดงเฉพาะชั้นนั้น) → เลือกวันที่ → แตะเลือกประเภทความดี → เลือกชื่อครูผู้บันทึก → กดบันทึก
  5. ประเภท 💰 เก็บเงินได้: กรอกจำนวนเงิน ระบบจะคำนวณคะแนนให้อัตโนมัติตามเกณฑ์ 2 บาท = 1 คะแนน
  6. สามารถแนบรูปภาพประกอบได้ 1-2 ภาพ (ไม่บังคับ) เช่น รูปเงินที่เก็บได้ หรือรูปขณะทำความดี
  7. คะแนนความดีจะสะสมเข้าตัวนักเรียนอัตโนมัติ ดูอันดับได้ที่ 📊 รายงานสรุป → Top 10
  8. แก้ไข/ลบรายการที่บันทึกผิดได้ที่ 📜 ประวัติความดี (ระบบจะปรับคะแนนสะสมให้ถูกต้องเอง)
  9. ส่งออกรายงานเป็น Excel หรือสั่งพิมพ์ได้ที่ 📊 รายงานสรุป และสำรองข้อมูลได้ที่ ⚙️ ตั้งค่าระบบ → สำรองข้อมูล

🚀 วิธี Deploy เป็น Web App (สำหรับครูที่นำโค้ดไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → เมนู Extensions → Apps Script
  2. วางโค้ดไฟล์ Code.gs ลงในไฟล์สคริปต์หลัก
  3. กด + → HTML ตั้งชื่อไฟล์ว่า index แล้ววางโค้ดหน้าเว็บทั้งหมด
  4. กด Deploy → New deployment → Web app ตั้งค่า Execute as: Me และ Who has access: Anyone (หรือ Anyone with Google account)
  5. กด Deploy อนุญาตสิทธิ์ (Authorize) ตามที่ระบบถาม แล้วคัดลอก Web App URL ไปแชร์ให้คุณครูใช้งาน
  6. ⚠️ สำคัญ: หากแก้ไขโค้ดภายหลัง ต้อง Deploy เวอร์ชันใหม่ทุกครั้ง (Deploy → Manage deployments → ✏️ → Version: New version → Deploy) การกดบันทึกโค้ดอย่างเดียวจะไม่อัปเดตเว็บจริง
